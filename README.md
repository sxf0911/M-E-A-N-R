## Requirements and Environment
* Node.js
* Express 4
* Mongodb
* AngularJS
* Redis

## Installation

### 1. Clone the repo

    git clone https://github.com/icyse/mean-blog.git

### 2. NPM install

    cd mean-blog
    npm install
    
### 3. Database stuff

	vi config.js

Ajust your Database config and Redis config

### 4. Start the server

	npm start
